Armor FAQ

Q: What are the armor sets?
4 sets: Shadow Set (5 plate, Hernand), Fallen Kingdom Set (5 mixed, Hernand), Dark Ringleader Set (3 cloth/leather, Demeniss), and Iron Plate Set (4 craftable). Browse all sets →
Q: What's the best armor?
Dark Ringleader Set has the highest DEF (18–21). For plate, Fallen Kingdom Set balances defense and coverage. Use the Armor Tier List → for data-driven rankings.
Q: How do I get armor?
Crafting (Iron Plate Set), location finds (Shadow & Fallen Kingdom in Hernand), boss drops (Dark Ringleader in Demeniss), and vendors (Scout Leather Vest, Mystic Cloth Robe).
Q: What do the 5 armor slots do?
Head, Chest, Gloves, Boots, and Cloak — each provides DEF. Chest gives the highest DEF. Cloaks give least but often have unique bonuses. All slots can socket Abyss Gears.
