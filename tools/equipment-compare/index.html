About This Tool

The Equipment Compare tool lets you compare any two weapons from our database side by side. All seven core stats are displayed: Attack, Defense, Attack Speed, Critical Rate, Movement Speed, Stamina Regen, and Spirit Regen. The weapon with the higher value for each stat is highlighted in green.

Use this tool when deciding between weapon upgrades, comparing different weapon types (1H vs 2H), or evaluating how a weapon switch changes your overall build stats. Pair with the Build Calculator for a complete build analysis.
